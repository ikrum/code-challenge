# Coding Challenge
```
git clone https://github.com/ikrum/code-challenge.git
```

## Backend Problem
### Tic Tac Toe Game
```
cd tic-tac-toe
npm install
node index.js
```

## Frontend Problem
### String compression
```
cd compress
node compress.js
```
### URI comparison
```
cd url-parser
node index.js
```
### Task Management App
```
cd task-manager
bower install

// to run the app you can use any serve you like.
// to install serve
sudo npm install serve -g

// run the App
serve

// Please visit http://localhost:3000 and try this out.
```
